Lululemon Blog

Lululemon's previous blog needed an overall new design. I was provided with the design and I was able to implement it in a very tight deadline. Within 6 weeks, a much more appealing blog was up and running, content was migrated and everyone was very happy with the outcome.

Some of the features of this project are pinterest like responsive layout, filters and social media integration.

Technologies used on this project: Wordpress, HTML, Jquery, CSS, MySql, Omniture and Google Analytics.
