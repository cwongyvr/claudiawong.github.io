Seawheeze - Lululemon

Seawheeze is Lululemon's Half Marathon yearly event. The purpose of this project was to take the previous year's website and make hard-coded content to be managed content. By doing so, we allow content editors to easily update its content. I also added tracking capabilities to it.

Technologies used on this project: Wordpress, HTML, Jquery, CSS, MySql, Omniture and Google Analytics.
