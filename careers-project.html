Careers - Lululemon

The Careers Lululemon project consisted of building some webpages and adding filters to the current site.

The main task for this project was to allow people to browse through jobs easily and to allow them to filter jobs by Country, Province, City and Store. I built a plugin that would allow the jobs to be updated in the backend by using Ajax.

The technologies used on this project: Wordpress, HTML, Jquery, CSS, MySql, Omniture and Google Analytics.
